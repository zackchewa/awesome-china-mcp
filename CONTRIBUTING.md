# Contributing

Thanks for helping keep this index useful!

## Adding a server
1. Find the right category in `README.md` (Maps, Content, Office, Cloud, Finance, etc.).
2. Add one table row:
   - App / service name (中文 + English if helpful)
   - Link to the repo or official docs
   - `🏢` if official/first-party, `👥` if community
   - A short, factual one-line note (what it does)
3. Only add servers that are real and working. No vaporware, no dead links.
4. Prefer the official server first, then list strong community alternatives by stars.

## Removing / fixing
Dead repo, renamed project, or wrong category? PRs to fix are just as welcome as additions.

## Style
- Keep notes short and neutral.
- Star counts are approximate and will drift — don't sweat exact numbers.
